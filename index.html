&commands
Description: Lists all available commands.
Usage Example: &commands
Reply: @user → &highestrank, &record, &recordbydivision, &rikishi, &shansho, &versus, &kimaritehistory, &bashoresults. For a comprehensive list of commands and usage examples visit www.sumo-orb.com
&highestrank
Description: Shows the highest rank ever achieved by a rikishi.
Usage Example: &highestrank onosato
Reply: @user → Yokozuna 1 East
&record
Description: Shows the overall record for a rikishi.
Usage Example: &record onosato
Reply: @user → Onosato | Wins: 170 | Losses: 54 | Matches: 224 | Basho: 17 | Yusho: 5
&recordbydivision
Description: Shows a rikishi's record by division.
Usage Example: &recordbydivision onosato
Reply: @user → Onosato | Makuuchi: 136-44 | Makushita: 10-4 | Juryo: 24-6
&rikishi
Description: Shows info about a rikishi.
Usage Example: &rikishi onosato
Reply: @user → Name: Onosato | Rank: Yokozuna 1 West | Heya: Nishonoseki | Birth: 2000-06-07 | Shusshin: Ishikawa-ken, Kahoku-gun, Tsubata-machi | Height: 192cm | Weight: 183kg | Debut: 202305
&shansho
Description: Shows the number of special prizes (sansho) a rikishi has won.
Usage Example: &shansho onosato
Reply: @user → Onosato | Gino-sho: 3 | Kanto-sho: 3 | Shukun-sho: 2
&versus
Description: Shows the head-to-head record and last bout between two rikishi.
Usage Example: &versus onosato oho
Reply: @user → Onosato 6 - 3 Oho. Their last bout was at 2025-11 on day 12. Onosato won by yorikiri.
&kimaritehistory
Description: Shows the last recorded match where a specific kimarite (winning technique) was used, including winner, opponent, basho, day, and bout number.
Usage Example: &kimaritehistory uchimuso
Reply: @user → Last recorded Uchimuso: Aonishiki defeated Kotozakura on 2025-11 day 15, bout 24.
&bashoresults
Description: Shows the yusho (division winners) and special prize winners for a given basho.
Usage Example: &bashoresults 2025-11
Reply: @user → Yusho: Aonishiki (Makuuchi), Fujiryoga (Juryo), ... | Shukun-sho: Aonishiki | Kanto-sho: Kirishima, Ichiyamamoto | Gino-sho: Aonishiki, Yoshinofuji
